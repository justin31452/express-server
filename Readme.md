Express-Server for MHS
===

## Installation

> build at FreeBSD 12.1-RELEASE r354233 GENERIC 

1. Install Nodejs.
2. Install pm2 $npm install pm2
3. Clone repository.
4. cd to express-server.
5. Run $npm install.
6. Run $pm2 start bin/www --watch --max-memory-restart 20M.
